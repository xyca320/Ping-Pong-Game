# Replica of the Legendary Ping-pong Game by Eason :)

...with Chill Musics and Smashing !

----------------------------------------------------
**Install:**
1. First Zip-up all the files, and use a repo template for Raylib C++ like such - https://github.com/educ8s/Raylib-CPP-Starter-Template-for-VSCODE-V2
2. Simply open up the zip and extract the folder "src" and "assets" by drag and dropping it into the Template.
3. Then download the latest Raylib Library from their website - https://www.raylib.com/
4. Finally simply double click the main.exe and enjoy the chillest replica of the Ping-Pong Game.
